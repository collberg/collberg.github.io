  • New grant. Tigress is supported by a new $950,240 NSF grant 2040206 SaTC:TTP:Medium: The Tigress Endpoint Protection Tool.
  • Waleed M. will be joining the project as a senior software engineer!
  • ARM64 support: In the upcoming version 3.3 Tigress will have partial support for Apple's Darwin/M1.
  • Checksum transformation.  New in the upcoming version 3.3.
  • Inline transformation:  New in the upcoming version 3.3.
  • Plugins:  New in the upcoming version 3.3. This allows you to, for example, add your own opaque predicates.
  • Optimize transformation: New in the upcoming version 3.3. Standard intra-procedural optimizations, useful, for example, after inlining.
  • SoftwareMetrics transformation: New in the upcoming version 3.3. Implements standard Software Complexity Metrics.
  • Function regions. Some transformations now support function regions, i.e. a transformation can be applied to a part of a function, rather then the whole one.
  • Json support. We are slowly rolling out support for Json throughout Tigress. For example, the option <tt>--jsonOut=filename.json</tt> will generate the output program as an abstract syntax tree (AST) in Json format. This can be useful for forther processing of the obfuscated code.
