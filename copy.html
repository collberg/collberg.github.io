Copy

free web templates

This transformationssimply makes a copy of a function. This can be useful with the Merge for example, making it possible to merge a function with itself.

OptionArgumentsDescription
--Transform Copy Make a copy of a function.
--CopyName string Name of the new function. Default=Current name followed by '_' and a random number.
