Windows Installation

The Tigress Windows system consists of two parts, the Tigress obfuscator itself and a Visual Studio plugin. This allows you to run Tigress either from the command line or from within Visual Studio.


Download and Install Tigress

Download the Windows install file from the download page and follow the steps below.

Let's assume you've downloaded the install file in the Downloads folder:

Double click on the install file:

If you already have Perl install you can uncheck this box:

Otherwise, install Perl:

Click Next:

Tigress is normally installed in the location C:\Program Files\University of Arizona\Tigress C Source Code Obfuscator\Tigress.

It's best to keep the default install location, but if you do decide to select another location, you will need to update the Tigress BAT script, YOUR_INSTALL_LOCATION\tigress.bat. Change the permissions of the tigress.bat file so that you can edit it, open it up in an editor, and set the TIGRESS_HOME variable to the directory where you installed Tigress.

Click Install:

Click Finish:

Click Install to install the Visual Studio plugin:

Click Close to finish the installation:


Running Tigress from the command line

Here's an example of how you run Tigress from the command line.

1

Run the vcvars64.bat script.

2

Create a C program foo.c.

3

Compile foo.c with cl and run it to make sure it's a valid C file (Tigress will not work on incorrect C programs).
